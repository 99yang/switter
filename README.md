# 주요 기능
### 로그인, 회원가입
- 이메일 형식과 github 연동 지원
- 비밀번호 재설정시 비밀번호변경 이메일 전송

### 트윗
- 사진 첨부 기능 (1mb 미만으로 제한)
- useEffect hook사용 다른 화면이거나 로그아웃 상태일 때 이벤트 리스너 발동 x
- 비용절약으로 트윗 리미트 25개 제한
